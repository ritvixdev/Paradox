1. Selectors in JS help to get specific elements from

variable

4. const can be assigned only once, and its

page as a tree-like structure that allows JavaScript to
dynamically access and manipulate the content and
structure of a web page.

that performs a specific task.

7.Arrow functions, also known as fat arrow

defining functions in JavaScript.

8.An array is a data type that allows you to
store multiple values in a single variable.
